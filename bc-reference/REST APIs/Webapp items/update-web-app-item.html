Update Web App Item

Update a web app item. The update can be full (sending all properties), or partial (sending just the ones that need updating). Sending a property with a null value or not sending it at all has the same effect: it will not be changed.

Request

  • Method: PUT
  • Server: https://[app key here]-[site_ID here]-apps.worldsecuresystems.com. Take a look at the Authorize your API calls document for more info on how this URL is formed.
    • Note: When building apps always use relative request URLs. Do not use the "full" URL above because you might have problems running your app on a different site as the site_ID parameter will be different.
  • Path: /api/v2/admin/sites/[siteId]/webapps/[webAppName]/items/[webAppItemId]
    • Alternatively, use "current" instead of [siteId]
  • Authorization header: This should contain the authorization token. Here is how to obtain the token.
  • Required Permissions: Edit Web App Items

Parameters

Body - A WebAppItem object containg the following properties:

  • {% raw %}name{% endraw %} - the name of the web app item (string)
  • {% raw %}weight{% endraw %} - the weight of the web app item (the order in which they will appear in frontend) (integer)
  • {% raw %}releaseDate{% endraw %} - the release date of the web app item; uses the format yyyy-mm-dd (string)
  • {% raw %}expiryDate{% endraw %} - the date when the web app item will expire; uses the format yyyy-mm-dd (string)
  • {% raw %}enabled{% endraw %} - a boolean indicating whether this web app item is enabled or disabled (boolean)
  • {% raw %}slug{% endraw %} - the suffix that will be used to build its frontend URL. This is appended to the URL of the webapp. (string)
  • {% raw %}description{% endraw %} - the description or content of the webapp item (string)
  • {% raw %}roleId{% endraw %} - the id of the role assigned to this web app item, if any (integer)
  • {% raw %}address{% endraw %} - the address of this webapp item if any (string)
  • {% raw %}city{% endraw %} - the city of this webapp item if any (string)
  • {% raw %}state{% endraw %} - the state of this webapp item if any (string)
  • {% raw %}zipCode{% endraw %} - the zip code of this webapp item if any (string)
  • {% raw %}country{% endraw %} - the country of this webapp item if any. It should be the 2 letter country code returned by the Countries API (https://zerowing.corp.adobe.com/display/BusinessCatalyst/System+API+reference#SystemAPIreference-GetCountriesList) (string)
  • {% raw %}fields{% endraw %} - an object storing the webapp item's custom fields. This field is represented in 2 ways:
    • in json, each custom field is a property of this object
    • in xml, it is an array of objects having the following properties:
      • {% raw %}name{% endraw %} - the name of the custom field
      • {% raw %}value{% endraw %} - the value of the custom field. It can be any of the supported types: number, string, boolean

Response

Returns empty body with a header response code.

Accepts and returns JSON as Content-Type.

Error codes

This method will return the following error codes:

  • {% raw %}200{% endraw %} - success
  • {% raw %}401{% endraw %} - unauthorized - when the authentication token is incorrect
  • {% raw %}403{% endraw %} - forbidden - when the user does not have the proper permissions
  • {% raw %}404{% endraw %} - not found
    • {% raw %}190001{% endraw %} - The web app could not be found (the webAppName param from the URL does not match)
    • {% raw %}200001{% endraw %} - The webapp item was not found (the webAppName and webAppItemId from the URL do not match)
  • {% raw %}400{% endraw %} - bad request; sub-error codes:
    • {% raw %}200000{% endraw %} - An unspecified error has occured
    • {% raw %}200002{% endraw %} - The slug is invalid
    • {% raw %}200003{% endraw %} - Using this slug will make the entire frontend URL larger than the maximum allowed length
    • {% raw %}200004{% endraw %} - The slug is already used
    • {% raw %}200005{% endraw %} - The name provided is invalid (empty or null)
    • {% raw %}200006{% endraw %} - A custom field failed validation
    • {% raw %}200008{% endraw %} - There is a problem with the address. Either the webapp does not allow storing addresses, or one of the address fields is empty
