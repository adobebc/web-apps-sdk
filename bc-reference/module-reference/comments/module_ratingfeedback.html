{module_ratingfeedback}

Description

Displays user comments on an item.

  • New named parameters documentation
  • Legacy module documentation

Syntax

{module_ratingfeedback render="item" collection="my_custom_collection_name" sortType="datereverse|date" template="/folder/template.tpl"}

Note: When using named parameters always wrap the individual parameters in double quotes.

Parameters

  • render - for this module this can only be item. For more information on how to use this parameter take a look at the render parameter's article for more details on how to use it
  • collection - use this if you need to access the module's Liquid output tags
  • sortType - if this parameter is present the comments are ordered newest first
  • template - render the module using a custom template. You can leave the value of the template parameter blank to suppress the original output and use Liquid output tags instead

This module can be rendered using:

  • the comment layout - on this layout you can use either the legacy tags or Liquid output tags. Here is a list of all the tags you can use the Comment layout reference page
  • your own custom layout - take a look at the custom templates article for more details
  • Liquid collections - here are the Liquid output tags you can use:
Liquid tag Legacy tag Description
{{country}} {tag_country}Country of the person who provided the comment
{{body}} {tag_feedback}Customer comment
{{flagUrl}} {tag_flag}Image of the flag of the country of the person who provided the comment
{tag_fullname}Displays the full name of the person who provided the comment.If the "Website" field was not blank when the comment form was submitted the name is rendered as an anchor. If the "Website" field was blank at the time the comment form was submitted the output is in plain text. If the comment was submitted without having the "Name" field filled in "Anonymous" is displayed.
{{fullName}} {tag_fullname_nolink}Full name in plain text of the person who provided the comment. If the comment was submitted without having the "Name" field filled in "Anonymous" is displayed.
{{rating}} {tag_rating}Rating in stars
{tag_usagedate} Date when comment was given in this format - 27-Mar-2014 12:02
{{website}} {tag_website}Website of the person who provided the comment. If the comment form was submitted with the "Website" field not filled in this tag will output "{tag_website}"

Examples

  • {module_ratingfeedback sortType="date"} - displays the comments newest first
  • {module_ratingfeedback template="/layouts/custom/comments.tpl"} - displays the comments using the comments.tpl custom template
  • {module_captchav2 collection="collection1" template="/folder/template.tpl"} - in this case nothing will be displayed, however the module's Liquid tags will become available in the collection1 collection.

Syntax

{module_ratingfeedback,date template="/customtemplate.tpl"}

Parameters

  • date - if this parameter is present the comments are ordered newest first
  • template - the path to the custom template you want this module to be rendered with

Layouts
