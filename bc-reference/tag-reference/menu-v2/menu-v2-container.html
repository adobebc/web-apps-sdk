Module menu V2 - container.html

Is used by these modules

  • {module_menu}

Accessible from

Unlike other layouts the module_menu v2 layouts can only be accessed in the "Develop" section of the site or if you connect via sFTP at this path - /ModuleTemplates/Menu/MENU_NAME/container.html

This is a wrapper around the actual menu output. You can add any additional mark-up for the menu here. For example, If you want to use a widget, this is where you would add resources such as JavaScript/CSS files or widget initialization code. This file needs to have the {% raw %}{tag_menugroup}{% endraw %} in order to render the menu.

Renders these tags

{% raw %}{tag_menugroup}{% endraw %}Top level tag which resides inside container.html
