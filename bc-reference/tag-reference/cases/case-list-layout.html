Renders these tags

Liquid tag Legacy tag Description
{{assignedTo}} {tag_assignedto}Customer service representative the case is assigned to e.g. John Smith
{{caseNumber}} {tag_casenumber}Case number
{{counter}} {tag_counter}Incremental count of the item on the page. For several cases attached to a contact in the CRM,  when using {module_case,a,,Default} (sorted by create date), the case created most recently is listed with counter "1". All that follow display a counter with an increment of 1.
{{createdate}} {tag_createdate}Date and time the case was first created e.g. 25-Mar-2013 12:13 PM
{{status}} {tag_status}Status of the case, possible values are New, Open, Escalated, Closed, Re-Opened
{{subject}} {tag_subject}Subject of the case - formatted as a link, pointing to the detail layout
{{ownerName}} {tag_ownername}Full name of the owner of the case e.g. John Smith
{{lastUpdateDate}} {tag_lastupdatedate}Date and time the last message was received (Ex. 25-Mar-2013 12:13 PM
{{workflow}} {tag_workflow}Workflow being used by the case. It will not output anything if no workflow is present
