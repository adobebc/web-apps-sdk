{module_captchav2}

Description

Displays the CAPTCHA verification image and the input text box. This module can be added from the webform detail panel like so: http://helpx.adobe.com/business-catalyst/using/create-insert-form.html#id_33012

This module also supports Liquid, take a look at this article to learn how to access its Liquid output tags and render it using your own custom logic.

  • New named parameters documentation
  • Legacy module documentation

Syntax

{% raw %}{module_captchav2 backgroundColor="DimGray" foregroundColor="White" width="160" height="36" collection="my_custom_collection_name" template=""}{% endraw %}

Note: When using named parameters always wrap the individual parameters in double quotes. Also, do note that modules using named parameters do not work for campaigns just yet.

For background and text color, you can use named colors or hexadecimal color values. See HTML Color Picker on www.w3schools.com

Examples

  • {% raw %}{module_captchav2 backgroundColor="LightBlue" foregroundColor="#FF3300" width="100" height="36" }{% endraw %}

Parameters

  • collection - use this if you need to access the module's Liquid output tags
  • template - this module does not support custom templates. You can use the template parameter however to supress the module's output

Syntax

{% raw %}{module_captchav2,[background color],[text color],[width in pixels]}{% endraw %}

For background and text color, you can use named colors or hexadecimal color values. See HTML Color Picker on www.w3schools.com

Examples

  • {% raw %}{module_captchav2,LightBlue,#FF3300,220,36}{% endraw %}
