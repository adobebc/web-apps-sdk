Order_PaymentProcess

Process a payment for an order via a payment gateway.

Warning: This API endpoint will eventually be deprecated (existing implementations will still continue to work). We strongly advise to use the new Order v3 REST API endpoints. Here is an example on how to get the list of orders created after a specified date using the new endpoints.

Request

  • Method: SOAP
  • Server: https://[app key here]-[site_ID here]-apps.worldsecuresystems.com. Take a look at the Authorize your API calls document for more info on how this URL is formed.
  • Path: /catalystwebservice/catalystcrmwebservice.asmx
  • The username and password fields:You can make SOAP API calls can be authorized either by using the actual username or password of an Admin username or, if you are making the calls from an app leave the username field empty and use the authorization token as the password.

Parameters

  • {% raw %}siteId{% endraw %} - ID of the site (integer)
  • {% raw %}username{% endraw %} - email address of user account, leave empty if using site token (string)
  • {% raw %}password{% endraw %} - password of user account, or site authentication token for specified site (string)
  • {% raw %}orderId{% endraw %} - (integer)
  • {% raw %}emailInvoiceToCustomer{% endraw %} - (boolean)
  • {% raw %}creditCardInfo{% endraw %} - (object)
    • {% raw %}CardNo{% endraw %} - (string)
    • {% raw %}CardHolder{% endraw %} - (string)
    • {% raw %}ExpiryMonth{% endraw %} - (string)
    • {% raw %}ExpiryYear{% endraw %} - (string)
    • {% raw %}CardType{% endraw %} - (integer)
    • {% raw %}CardCCV{% endraw %} - (string)
    • {% raw %}IssueMonth{% endraw %} - (string)
    • {% raw %}IssueYear{% endraw %} - (string)
    • {% raw %}IssueNumber{% endraw %} - (string)
    • {% raw %}StoredCardToken{% endraw %} - (string)
  • {% raw %}transactionInfo{% endraw %} - (object)
    • {% raw %}RequestParams{% endraw %} - (object)
    • {% raw %}Description{% endraw %} - (string)
    • {% raw %}PaymentReference{% endraw %} - (string)
    • {% raw %}Amount{% endraw %} - (double)

To use a site token instead of username/password, send an empty {% raw %}username{% endraw %} field and the site token as the {% raw %}password{% endraw %}. See example below.

Response

A Order_PaymentProcessResponse object with the following properties:

  • {% raw %}Order_PaymentProcessResult{% endraw %} - (boolean)

Examples
