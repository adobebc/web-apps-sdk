Receipt (buy) layout

Accessible from

  • Admin Console: Site Manager > Module Templates > Online Shop Layouts
  • SFTP & Develop Mode: /Layouts/OnlineShop/order_receipt-XX.html

Replace "XX" with the country code you have added in the Site Settings > Site Domains panel (see the "Country" column). 

If you have multiple domains with different "Country" values you will find multiple order-receipt-XX files, one for each country. This way, with one layout for every country defined you can vary the contents of the layout to accomodate a multilingual website.

Renders these tags

Liquid tag Legacy tag Description
{{orderId}} {tag_orderid}Internal ID associated with the purchase
{{amount}} {tag_amount} The amount paid
{{authorizationCode}} {tag_authorizationcode} Authorization code returned by the payment gateway
{{errorMessage}} {tag_errormessage} Error message returned by the payment dateway
{{invoiceNumber}} {tag_invoicenumber} Invoice number
{{paymentStatus}} {tag_paymentstatus} The status of the order, possible values Success, Failed or Pending
{{amountExceptShipping}} The total amount excluding shipping cost
