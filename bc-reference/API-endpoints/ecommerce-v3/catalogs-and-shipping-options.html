Catalogs and shipping options

    List the shipping options marked to exclude catalog

    Request

    • Method: GET
    • Version: v3, this API call supports additional sort and filter mechanisms as well as retrieving only a sub-set of fields

      Note: In order to access the v3 API endpoints please make sure you have enabled the new rendering engine.

    • Server: https://[app key here]-[site_ID here]-apps.worldsecuresystems.com. Take a look at the Authorize your API calls document for more info on how this URL is formed.
      • Note: When building apps always use relative request URLs. Do not use the "full" URL above because you might have problems running your app on a different site as the site_ID parameter will be different.
    • Path: /webresources/api/v3/sites/current/catalogs/[catalog_ID]/shippingoptions
      • Alternatively, use use siteID instead of 'current'
    • Authorization header: This should contain the authorization token. Here is how to obtain the token.

    Response

    An array of the first 10 discount codes; each object has the following properties:

    • {% raw %}id{% endraw %} - the id of the shipping option (integer)
    • {% raw %}siteId{% endraw %} - the id of the site (integer)
    • {% raw %}"description"{% endraw %} - the shipping option's description (string)
    • {% raw %}"price"{% endraw %} - (integer)
    • {% raw %}"type"{% endraw %} - the type of shipping option (integer)
      • 1 - User defined
      • 2 - Australia Post
      • 3 - FedEx
      • 4 - USPS
      • 5 - Free shipping
      • 6 - DHL
      • 7 - UPS
    • {% raw %}"packageType"{% endraw %} - (integer)
    • {% raw %}"taxCodeId"{% endraw %} - (integer)
    • {% raw %}"countryCode"{% endraw %} - (string)
    • {% raw %}"state"{% endraw %} - (string)
    • {% raw %}"postcode"{% endraw %} - (string)
    • {% raw %}"productMinWeight"{% endraw %} - (integer)
    • {% raw %}"productMinVolume"{% endraw %} - (integer)
    • {% raw %}"productMinHeight"{% endraw %} - (integer)
    • {% raw %}"productMinDepth"{% endraw %} - (integer)
    • {% raw %}"productMaxWeight"{% endraw %} - (integer)
    • {% raw %}"productMaxVolume"{% endraw %} - (integer)
    • {% raw %}"productMaxHeight"{% endraw %} - (integer)
    • {% raw %}"productMaxDepth"{% endraw %} - (integer)
    • {% raw %}"totalOrderPriceMin"{% endraw %} - (integer)
    • {% raw %}"totalOrderPriceMax"{% endraw %} - (integer)
    • {% raw %}"handlingBaseCharge"{% endraw %} - (integer)
    • {% raw %}"handlingPerItemCharge"{% endraw %} - (integer)
    • {% raw %}"freeThreshold"{% endraw %} - (integer)
    • {% raw %}"limitDeliveryTo"{% endraw %} - (string)
    • {% raw %}"userAccountNumber"{% endraw %} - (string)
    • {% raw %}"userAccountId"{% endraw %} - (string)
    • {% raw %}"userAccountKey"{% endraw %} - (string)
    • {% raw %}"userAccountPassword"{% endraw %} - (string)
    • {% raw %}"upsAccountType"{% endraw %} - (integer)
    • {% raw %}"upsPackageType"{% endraw %} - (integer)
    • {% raw %}"upsConfirmationType"{% endraw %} - (integer)

    Accepts and returns JSON as Content-Type.
