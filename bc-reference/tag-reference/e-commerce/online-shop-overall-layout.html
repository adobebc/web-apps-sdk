{tag_cataloguelist}

Lists the sub-catalogues in the current catalogue, for example {tag_cataloguelist,x} where x is the number of catalogues per row).

The full syntax is {tag_cataloguelist, rowLength, targetFrame, notUsed, sortType, hideEmptyMessage, list/table}

Change: {tag_cataloguelist,4} to {tag_cataloguelist,4,,,,,true} to display in an unordered list structure.

{tag_productlist}Lists the products in the current catalogue. Use {tag_productlist,u,v,w,x,y,z} where:
  • u is the number of products per row
  • v is the target frame, e.g. _blank or leave empty
  • w is the number of products per page (limit of 500 products per page)
  • x is the sort type, e.g. Price or Alphabetical
  • y allows you to hide empty message, e.g. This catalog has no products
  • z is list type, for example change {tag_productlist,4} to {tag_productlist,4,,,,,true} to display in an unordered list structure.
For additional information on how to render the output of this tag using Liquid take a look at the Enable other Liquid tags article.
